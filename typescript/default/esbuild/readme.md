## **⚙️ Setup & Build Instructions**  

### **1️⃣ Install Dependencies**  
Run the following command inside your plugin directory:  
```sh
npm install
```

### **2️⃣ Build the Plugin**  
Build Plugin.zip using:  
```sh
npm run build
```

### **3️⃣ Publish**  
Publish the builded aftifact to Acode's Plugin store:  
```sh
npm run publish
```
